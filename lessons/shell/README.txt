Material for introduction to the shell.

Heavily influenced by the Software Carpentry novice materials; these are the numbered lessons (00 through 06). The web versions can be found at:

http://software-carpentry.org/v5/novice/shell/index.html

The shell.md file is the specific lesson for Data Carpentry. Also see the cheatsheet in cheatsheet directory. 

The data files are:
* surveys.csv
* plots.csv
* species.csv

which come from Ernest et al 2009:
http://esapubs.org/archive/ecol/E090/118/metadata.htm
